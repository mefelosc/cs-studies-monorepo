# Guia de Solução de Problemas: Conectividade de Rede

## Descrição do Problema
O usuário relata incapacidade de acessar a rede interna ou a internet.

## Verificações Iniciais
1. Verifique se o cabo de rede está conectado.
2. Verifique se o Wi-Fi está ativado.
3. Verifique a atribuição do endereço IP (`ipconfig` no Windows, `ifconfig` ou `ip a` no Linux).

## Passos para Resolução
- Reinicie o adaptador de rede.
- Pingue o gateway para verificar a conectividade local.
- Pingue um DNS público (ex: 8.8.8.8) para verificar o acesso à internet.

## Escalonamento
Se o problema persistir, entre em contato com o Administrador de Rede.
