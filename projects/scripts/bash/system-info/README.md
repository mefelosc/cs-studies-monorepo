# Simple System Monitor

This is a simple Bash project that collects and displays basic system information.

## Features
- Operating System details
- Kernel version and Architecture
- System uptime and Load average
- Memory usage
- Disk usage (excluding virtual filesystems)
- Top CPU-consuming processes

## How to use
1. Give execution permission (if not already set):
   ```bash
   chmod +x system_info.sh
   ```
2. Run the script:
   ```bash
   ./system_info.sh
   ```
