# Salve Abas

Extensão de navegador para salvar e gerenciar abas facilmente.

## Como rodar a extensão

1. **Clone ou baixe este repositório.**
	- Certifique-se de ter todos os arquivos na mesma pasta.

2. **Abra o navegador (Google Chrome, Edge, Brave, etc).**

3. **Acesse a página de extensões:**
	- Chrome/Brave/Edge: `chrome://extensions/`

4. **Ative o modo de desenvolvedor** (canto superior direito).

5. **Clique em "Carregar sem compactação"** ou "Load unpacked".

6. **Selecione a pasta `salve-abas`** (onde estão os arquivos `manifest.json`, `index.js`, etc).

7. A extensão aparecerá na lista e estará pronta para uso!

---

### Dicas
- Para atualizar a extensão após alterações, clique em "Atualizar" na página de extensões.
- Consulte o `manifest.json` para detalhes de permissões e funcionamento.
